## Summary

コードブロックのシンタックスハイライトを上書きする Chrome 拡張です。

## Features

- 任意のテーマでのシンタックスハイライトの上書き
- 任意の言語でのプレビュー
- 除外ページの設定
- ダークモード対応

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
